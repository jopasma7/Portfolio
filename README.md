# Portfolio

Portafolio web personal con proyectos, experiencia, formación y contacto. Responsive, bilingüe (español/inglés) y con modo claro/oscuro.

## 🌐 Demo

[Ver online](https://jopasma7.github.io/Portfolio/)

## Tecnologías

- HTML5, CSS3, JavaScript
- Responsive Design
- Multilenguaje (ES/EN)
- Modo claro/oscuro

## Descripción

Este sitio muestra mi trayectoria profesional, proyectos destacados, habilidades técnicas y formación académica. Incluye formulario de contacto y enlaces a redes profesionales.

> Desarrollado y diseñado para destacar mi perfil profesional y facilitar el contacto con empleadores o colaboradores.
